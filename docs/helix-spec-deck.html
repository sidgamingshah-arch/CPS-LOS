Operational notes

  • SQLite is single-writer → Hikari pool size 1; audit joins the caller’s tx (no REQUIRES_NEW).
  • Inter-service calls degrade gracefully — config-service down ⇒ conservative built-in packs (v0, visible in trace).
  • All write endpoints accept X-Actor; recorded in the immutable trail.

1 · Regulatory abstraction layer PRD §10

A new regime is overlay data, not a code branch.

Composition

  • JurisdictionProfile — capital approach, CVA approach, provisioning frameworks, reported-provision policy, residency, reporting pack.
  • RulePack — (code, type, jurisdiction, version) + JSON payload + dual sign-off (policy + model risk) before activation.

Pack types consumed downstream

CAPITAL_SA · ECRA_MAPPING · RATING_PD_MAP · LGD_MAP · PROVISIONING · DOA_MATRIX · CDD_TIERS · EXPOSURE_LIMITS · PRICING

Two regimes seeded

  • IN-RBI — SA Directions 2026, IRAC, max(ECL, IRAC), BA-CVA, DD-driven RW.
  • AE-CBUAE — Basel III, IFRS-9 only, SA-CVA.

Same CapitalEngine / EclEngine — different numbers, purely from the active pack.

GOVERNANCE · VERIFICATION · ROADMAP

Governed AI, named humans, examiner-ready trace

5 / 5

AI governance contract PRD §6, §11

  • Autonomy levels in code: A C D.
  • No GenAI in the deterministic path (capital / ECL / RWA). AI explains a number, quoting engine values.
  • Confidence routing on doc classification; low-conf → human queue.
  • Notch-limited rating overrides; feed the override-rate model-fit signal (alert > 25%).
  • EWS flags but never re-stages; staging is a named human action.
  • Copilot refuses credit-consequential imperatives; routes them to the gated workflow.
  • Immutable audit · actorType ∈ {HUMAN, AI, SYSTEM} on every event.

Figure → source → version trace PRD §13

  • SpreadCell · sourceDocument / page / coordinates / confidence; override audited.
  • CapitalResult.trace · CCF, base RW, DD uplift, applied RW, CRM, RWA, capital ratio + rule-pack code & version.
  • EclResult.trace · stage drivers, PD, LGD, overlay, IRAC class & rate, reported-provision policy + pack version.
  • Decision · routing rule applied, deviations, named decider, conditions.
  • One-click subject trace via /api/audit/subject?type=Application&id=….
